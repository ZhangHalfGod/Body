# 01 · 体态自测与功能评估

> 综合自原 [02-体态专项-肱骨前移与圆肩](../02-体态专项-肱骨前移与圆肩.md)、[04-中下斜方肌与菱形肌](../04-中下斜方肌与菱形肌.md)

## 1. 静态观察

### 1.1 侧面观

自然站立，从侧面拍照或观察镜子：

- **耳—肩峰—大转子** 是否大致一条线？
- 肩峰是否明显在耳垂前方？（→ 圆肩倾向）
- 肱骨头是否在肩峰前方？（→ 肱骨前移倾向）

### 1.2 背面观

- 肩胛内侧缘是否过于贴近脊柱？（→ 肩胛内旋）
- 是否有翼状倾向——放松时肩胛内侧缘翘起？（→ 前锯肌弱）
- 两侧肩胛高度是否对称？

## 2. 简单功能测试

### 2.1 墙滑（Wall Slide）

| 项 | 说明 |
|----|------|
| **做法** | 背贴墙，屈肘 90°，手掌+手腕贴墙，沿墙上下滑动 |
| **正常** | 手能平滑滑到头顶以上，腰不离墙，不耸肩 |
| **异常提示** | 腰离开墙、耸肩、手无法贴墙→中下斜方/肩胛上旋受限 |

### 2.2 外旋耐力

| 项 | 说明 |
|----|------|
| **做法** | 屈肘 90° 靠墙或侧卧，轻阻力弹力带做肩外旋 |
| **正常** | 可持续 15+ 次无明显代偿 |
| **异常提示** | 很快力竭或肩峰前顶→肩袖外旋不足 |

### 2.3 肩胛后缩

| 项 | 说明 |
|----|------|
| **做法** | 站立，仅用肩胛骨向脊柱靠拢（不耸肩） |
| **正常** | 可明显内收 2-3 cm |
| **异常提示** | 只能耸肩、无法内收→中下斜方/菱形激活障碍 |

### 2.4 前锯肌测试（墙推）

| 项 | 说明 |
|----|------|
| **做法** | 双手推墙，观察肩胛骨 |
| **正常** | 肩胛平滑贴胸廓，内侧缘不翘起 |
| **异常提示** | 内侧缘/下角翘起→前锯肌弱（翼状肩胛） |

### 2.5 下斜方测试（Prone Y）

| 项 | 说明 |
|----|------|
| **做法** | 俯卧，双臂呈 Y 字（约 130° 外展），拇指朝上，双臂抬起 |
| **正常** | 可抬离地面 10-15 cm 且无耸肩 |
| **异常提示** | 上斜方明显绷紧、抬不高→下斜方激活差 |

## 3. 进度指标（怎么知道在好转）

| 指标 | 初始（需纠正） | 改善中 | 良好 |
|------|---------------|--------|------|
| **Wall slide 高度** | 手难过头、腰离墙 | 可过头、腰贴墙 | 手臂过耳、全程贴墙 |
| **静态肩胛内收维持** | < 5 秒 | 5-10 秒 | 15+ 秒不耸肩 |
| **划船酸感位置** | 二头/上背 | 肩胛内侧有感觉 | 肩胛内侧为主+背阔为辅 |
| **引体耸肩** | 明显 | 轻微 | 无 |
| **肩峰不适** | 过头动作有痛 | 偶有 | 无 |

## 4. 测试频率

- **全面自测**：每 4 周一次（与训练周期同步）
- **快速检查**：每次训练前（墙滑 + 肩胛后缩，1 分钟）
