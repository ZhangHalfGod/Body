# 04 · 动作质量控制与代偿识别

> 综合自原 [02-体态专项-肱骨前移与圆肩](../02-体态专项-肱骨前移与圆肩.md)、[04-中下斜方肌与菱形肌](../04-中下斜方肌与菱形肌.md)、[06-肌肉动作规划](../06-肌肉动作规划.md)

## 1. 核心原则

**动作质量 > 重量 > 组数** —— 错误动作练得越多，代偿越固化。

| 原则 | 说明 |
|------|------|
| **先收肩胛再动肢** | 所有拉类：先肩胛内收 2-3 cm，再屈肘 |
| **沉肩不耸肩** | 所有上肢动作中，颈部不应该变短 |
| **幅度完整但不牺牲控制** | 半程动作容易跳过薄弱角度 |
| **推拉配对** | 推日一定有拮抗拉类收尾 |

## 2. 常见代偿模式与纠正

### 2.1 划船类

| 错误 | 识别方法 | 后果 | 纠正 |
|------|----------|------|------|
| **耸肩 + 身体后仰** | 肩耸向耳朵，上斜方先酸 | 练不到目标，下背压力 | 减重 + 胸贴垫 + 先收肩胛 |
| **肘贴身体猛拉** | 肘紧贴肋骨两侧 | 背阔肌+二头代走 | 肘外展 45°-60° |
| **离心松掉肩胛** | 回放时肩胛先向前滑出 | 离心训练无效 | 全程保持肩胛微收 |

### 2.2 引体/下拉类

| 错误 | 识别方法 | 后果 | 纠正 |
|------|----------|------|------|
| **上拉时耸肩** | 杠/握把到位时耸肩最明显 | 上斜方+提肩胛代偿 | 拉前深吸气沉肩，拉时呼气 |
| **含胸拉** | 胸部内扣、肩峰前顶 | 圆肩加重 | 挺胸、肩胛后缩后拉 |
| **颈后下拉** | 握把拉到颈后 | 肩峰撞击风险 | 改颈前下拉或引体 |

### 2.3 卧推/推类

| 错误 | 识别方法 | 后果 | 纠正 |
|------|----------|------|------|
| **卧推耸肩** | 推起时耸肩、脖子变短 | 上斜方代偿、肩峰前顶 | 脚蹬地沉肩、肩胛后缩"挖坑" |
| **双杠含胸** | 下降时胸内扣、肩前顶 | 胸大肌+前束过度、前侧关节压力 | 稍前倾、肩胛下沉后缩 |
| **飞鸟内旋位** | 手臂内旋做夹胸 | 肩袖/前方关节压力 | 外旋位或缩短幅度 |

### 2.4 面拉类

| 错误 | 识别方法 | 后果 | 纠正 |
|------|----------|------|------|
| **双绳末端往两边扩展开** | 拉到耳前后继续往两侧水平外展 | 肱三头肌长头肌腱+后侧关节囊末端过载，后束刺痛 | 改为**拉到耳前→肘往后收（不是往两边）→停 1s→慢回** |
| **单绳 cue 直接套用到双绳** | 以为「扩展开=肩胛内收」 | 单绳上成立，双绳上变成纯水平外展 | 单绳「拉开」，双绳「肘往后收」 |
| **面拉耸肩** | 拉到最高点时颈部变短 | 上斜方代偿，中斜方/菱形未参与 | 降重，拉起前先沉肩锁定 |

### 2.5 错误循环示例

```
目标：菱形肌 / 中下斜方
         ↓
实际发生：
  上斜方耸肩 ──→ 提肩胛肌
  背阔肌主导 ──→ 肘贴身体猛拉
  腰/伸肌过伸 ──→ 假幅度
  肱二头 ──→ 先酸的是手臂
```

## 3. 推拉动作检查表

### 推前自检

- [ ] 弹力带外旋 2×15 已完成
- [ ] 肩胛后缩 2×10 已完成
- [ ] 卧推时脚踩地、臀不抬
- [ ] 全程不耸肩、颈不短

### 拉前自检

- [ ] Band Pull-apart 2×15 已完成
- [ ] Wall slide 或 Prone Y 2×10 已完成
- [ ] 划船：先收肩胛再拉肘
- [ ] 下拉：沉肩后拉，不耸肩

### 练后自检

- [ ] 推日：有做反向飞鸟/面拉 2 组
- [ ] 拉日：有做肩胛后缩激活
- [ ] 胸拉伸已完成

## 4. 重量进阶的条件

**不要仅因为"能做完组数"就加重。** 满足以下条件再考虑：

| 动作 | 进阶前必须满足 |
|------|----------------|
| 卧推 | 全程肩胛稳定、无耸肩、杠路顺畅 |
| 划船 | 肩胛先动、上斜方不先酸、无明显身体晃动 |
| 引体 | 耸肩 < 10%（视频自查） |
| 下拉 | 沉肩启动、胸触杠（宽握） |
| 深蹲 | 不圆背、膝盖轨迹稳定、深度达标 |

## 5. 推拉比例校对

| 训练日 | 推类组数 | 拉类组数 | 肩带纠正组数 |
|--------|----------|----------|-------------|
| 推日 | ~16（主项） | 0 | 2-4（反向飞鸟/外旋） |
| 拉日 | 0 | ~11（主项） | 2-4（激活+反向飞鸟） |
| 腿日 | 0 | 0 | 4（反向飞鸟） |

> 目标：每周拉类（含纠正）组数 ≥ 推类组数。
