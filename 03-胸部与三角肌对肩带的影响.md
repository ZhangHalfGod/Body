# 03 · 胸部与三角肌对肩带的影响

## 1. 胸大肌：不只是「练胸」

### 解剖要点

- **起点**：锁骨部、胸骨部、腹直肌鞘（部分）
- **止点**：肱骨大结节嵴（大结节间沟外侧）
- **主要动作**：肩内收、内旋、屈（锁骨部）

### 对肩胛与肱骨的影响

| 效应 | 机制 |
|------|------|
| **肱骨内旋** | 止点在肱骨前侧，缩短时持续内旋 → **肱骨前移** 风险 |
| **圆肩** | 与胸小肌、前三角协同，肩带整体前移 |
| **限制外旋与肩胛后倾** | 胸大肌过紧时，手臂难在外旋位抬举 |
| **卧推后的「含胸」** | 大重量推后胸大肌张力高，若不做拮抗拉类，姿态固化 |

### 训练注意

- **练胸不等于坏事**；问题是 **量与比例**：推多拉少、忽视拉伸与激活。
- 卧推时肩胛**后缩下沉**（「挖坑」）可部分保护，但若胸大肌/胸小肌已经短，仍可能代偿。
- **飞鸟、夹胸** 在内旋位做过多，对圆肩人群不友好；可略外旋位或缩短幅度。

## 2. 三角肌：三束分工与肩带

### 前束（Anterior Deltoid）

- **动作**：肩屈、内旋、水平内收（辅助）
- **与圆肩**：与胸大肌功能重叠；**过强 + 后束/外旋弱** → 肩峰前移
- **常见来源**：所有推类、前平举、Burpee、倒立撑

### 中束（Lateral Deltoid）

- **动作**：肩外展
- **肩带影响**：相对中性；但外展时若**上斜方代偿**，会加剧耸肩模式

### 后束（Posterior Deltoid）

- **动作**：肩伸、水平外展、外旋（辅助）
- **肩带影响**：与菱形肌、中下斜方、肩袖协同，**对抗圆肩**；但单独练后束**不能替代**肩胛内收肌

```
        前束 ──→ 内旋、屈     ═══╗
                              ║  盂肱关节
        后束 ──→ 伸、外展辅助  ═══╝
              ↑
        需配合：菱形肌、中下斜方、肩袖
```

## 3. 胸 + 肩 与 肩胛骨运动

### 肩胛前倾（Anterior Tilt）

- **拉动者**：胸小肌（见 05）、短缩的胸大肌下部
- **限制者**：下斜方、前锯肌（上旋时后倾）

### 肩胛内旋（Medial Rotation / 内侧缘远离脊柱）

- **关联**：胸大肌内旋肱骨时，常伴肩带内旋外观
- **纠正**：外旋训练 + 菱形肌/中下斜方内收，而非只练后束

### 耸肩（Elevation）

- **关联**：上斜方主导；推举、侧平举、引体若不会「沉肩」，前束/上斜方更占主导
- **要点**：主训练前用 **下斜方激活**（Y 字、Wall slide 沉肩）

## 4. 推拉比例与周期化建议

| 人群倾向 | 建议比例（组数/周） | 说明 |
|----------|---------------------|------|
| 圆肩 / 办公族 | 拉 : 推 ≈ **2 : 1** 或更高 | 水平拉 + 垂直拉 + 面拉 |
| 一般健身 | 拉 : 推 ≈ **1 : 1 ~ 1.5 : 1** | 视动作质量调整 |
| 力量举偏卧推 | 额外加 **面拉、Band pull-apart** | 不减少卧推，但补后链肩带 |

### 动作配对示例

- 卧推 ↔ 胸_supported 划船 / 面拉
- 过头推举 ↔ 引体 / _lat 下拉（宽握 + 全幅度肩胛）
- 双杠 ↔ 反向飞鸟 + 外旋

## 5. 实操：练胸/肩时如何「少伤害肩带」

1. **推之前**：弹力带外旋 2×15 + 肩胛后缩 2×10（无耸肩）
2. **卧推**：脚蹬地、肩胛后缩下沉；杠路略斜向脚侧（个体化）
3. **推举**：避免 excessive 颈后推举（外旋/撞击风险）；优先前侧推举 + 充分热身
4. **练后**：胸大肌拉伸 + **激活中下斜方**（见 04），而非只拉伸
