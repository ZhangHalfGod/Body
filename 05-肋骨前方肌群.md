# 05 · 肋骨前方肌群

## 1. 概述：为什么「肋骨前面」影响肩带

肩带不是悬在空中的——**肩胛基底贴在肋廓后外侧**。肋骨前方肌肉的紧张或呼吸模式异常，会改变：

- 肋骨角度（尤其 2–5 肋）
- 胸廓整体是「上抬」还是「三维扩张」
- 胸小肌、前锯肌的力学长度

这与 **圆肩、肱骨前移、翼状肩胛** 直接相关。

## 2. 主要肌肉

### 2.1 胸小肌（Pectoralis Minor）

| 项目 | 内容 |
|------|------|
| **附着** | 肋骨 3–5 → 肩胛**喙突** |
| **动作** | 肩胛**前倾**、**下旋**、**内旋**；辅助**吸气**（上提肋） |
| **与体态** | 短缩 → **圆肩**、肩胛前倾、可能压迫臂丛（胸廓出口综合征风险） |
| **与训练** | 卧推/伏案加重短缩；需松解 + 激活拮抗肌（下斜方、前锯） |

> 胸小肌是「把肩胛往前往下拉」的隐形推手，比胸大肌更常被忽视。

### 2.2 前锯肌（Serratus Anterior）

| 项目 | 内容 |
|------|------|
| **附着** | 肋 1–8/9 外侧面 → 肩胛**内侧缘**（前表面） |
| **动作** | 肩胛**外旋**、**上旋**、**前伸**（protraction） |
| **与体态** | 弱 → **翼状肩胛**；过头动作肩峰撞击风险上升 |
| **与菱形肌** | 前锯 **前伸/上旋** ↔ 菱形 **内收/下旋**，需**动态平衡**而非只练一边 |

### 2.3 肋间肌（Intercostals）

- **外肋间**：提肋，辅助吸气
- **内肋间 / 最内肋间**：降肋，辅助呼气
- **与体态**：长期 **浅快胸式呼吸** → 上胸肋 habitual 上抬，胸小肌、上斜方紧张，**膈肌参与不足**

### 2.4 膈肌（Diaphragm）

- 主呼吸肌；正确参与时胸廓应 **360° 扩张**（腹、肋、背侧）
- **与肩带**：膈肌功能差时，常用 **胸式 + 上斜方** 代偿吸气，间接恶化圆肩

### 2.5 腹直肌 / 腹外斜（与肋骨下缘相关）

- 过度卷腹、不含呼吸整合 → 肋骨 **下压不足** 或 **骨盆后倾**，影响胸廓-骨盆对位，间接限制肩胛基底

## 3. 呼吸模式与体态链

```
浅胸式呼吸（肋前上抬）
        ↓
胸小肌、上斜方、SCM 紧张
        ↓
肩胛前倾 + 内旋 + 难沉肩
        ↓
圆肩 / 肱骨前移 / 翼状肩胛
        ↓
中下斜方、前锯肌更难激活
```

### 改善方向（非医疗）

1. **鼻吸、慢吸**：感受肋骨向侧后方扩张，而非只抬胸
2. **呼气略长**：肋间肌与腹横肌参与，肋骨回落
3. **训练整合**：推拉前 2–3 分钟 **90-90 呼吸** 或 **鳄鱼式呼吸**

## 4. 针对性处理

### 4.1 胸小肌

- **松解**：网球或筋膜球压肋骨 3–4 与胸大肌深面；门框拉伸（臂略高）
- **激活拮抗**：Wall slide、Prone Y、前锯 push-up plus
- **避免**：长时间手臂外展 90° 以上办公无支撑

### 4.2 前锯肌

| 动作 | 要点 |
|------|------|
| **Scapular Push-up** | 仅肩胛前后，肘几乎直 |
| **Push-up Plus** | 俯卧撑顶端再「多推出去」一点 |
| **Wall Push Protraction** | 双手推墙，肩胛离开墙 |
| **Serratus Punch**（轻哑铃） | 臂伸直，只向前「戳」 |

### 4.3 肋间 / 膈肌

- **侧向呼吸练习**：手放肋骨两侧，吸气时推开手
- **死虫、鸟狗**：维持核心，呼吸不憋气
- 避免长期 **腰带束胸式** 健美卧推呼吸（非竞技需求不必）

## 5. 与中下斜方 / 菱形肌的协同

| 问题 | 肋骨前侧 | 肩胛后侧 |
|------|----------|----------|
| 圆肩 | 胸小肌短 | 菱形肌、中下斜方弱 |
| 翼状肩胛 | 前锯弱 | 下斜方弱 |
| 耸肩 | 上胸式呼吸 | 下斜方弱、上斜方紧 |

**合理顺序**：呼吸/胸小肌松解 → 前锯 + 中下斜方激活 → 主训练 → 拉伸胸大/小
