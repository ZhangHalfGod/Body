# 02 · 肩关节复合体生物力学

> 综合自原 [01-上半身体系概览](../01-上半身体系概览.md) 与 [03-胸部与三角肌对肩带的影响](../03-胸部与三角肌对肩带的影响.md)

## 1. 肩肱节律（Scapulohumeral Rhythm）

健康抬臂（0° → 180°）的典型分工：

| 阶段 | 角度范围 | 主导运动 | 关键肌肉 |
|------|----------|----------|----------|
| **初期** | 0°–60° | 主要盂肱关节运动 | 三角肌、肩袖 |
| **中期** | 60°–120° | 肩胛开始明显上旋 | 前锯肌 + 下斜方 |
| **末期** | 120°–180° | 盂肱 + 肩胛完全协同 | 需中下斜方/前锯充分参与 |

> **约 1/3 的抬臂幅度来源于肩胛上旋**。若中下斜方/前锯不足 → 上斜方代偿、耸肩。

## 2. 肩胛骨应有的运动

在健康动作中，肩胛骨应呈现：
- **上旋**（upward rotation）— 最重要
- **后倾**（posterior tilting）— 轻微
- **适度外旋**（external rotation）— 远离脊柱

任何一项缺失都会改变盂肱关节对位。

## 3. 胸肌与三角肌对肩带的力学影响

### 3.1 胸大肌

| 效应 | 机制 |
|------|------|
| **肱骨内旋** | 止点在肱骨前侧，缩短时持续内旋 → 肱骨前移风险 |
| **圆肩** | 与胸小肌、前三角协同，肩带整体前移 |
| **限制外旋与后倾** | 过紧时手臂难在外旋位抬举 |
| **卧推后的"含胸"** | 大重量推后胸大肌张力高，若不做拮抗拉类，姿态固化 |

### 3.2 三角肌三束

| 束 | 主要动作 | 体态关联 |
|----|----------|----------|
| **前束** | 肩屈、内旋 | 过强 + 后束/外旋弱 → 肩峰前移 |
| **中束** | 肩外展 | 若上斜方代偿 → 耸肩模式加剧 |
| **后束** | 肩伸、水平外展 | 与菱形肌、中下斜方协同，但**不能替代**肩胛内收肌 |

```
       前束 ──→ 内旋、屈     ═══╗
                                ║  盂肱关节
       后束 ──→ 伸、外展辅助  ═══╝
             ↑
       需配合：菱形肌、中下斜方、肩袖
```

### 3.3 推拉动作中的力矩

- **卧推**：若肩胛未后缩下沉 → 胸大肌+前束占绝对主导，缺乏后侧拮抗
- **过头推举**：若肩胛上旋受限 → 肩峰撞击风险升高
- **划船/引体**：若先动肘而非肩胛 → 力量由背阔肌+二头代走，中下斜方/菱形未参与
- **侧平举**：肩胛未沉 → 上斜方代偿（"假中束"）

## 4. 推拉比例与力学平衡

| 人群倾向 | 建议比例（组数/周） | 力学目标 |
|----------|---------------------|----------|
| 圆肩/办公族 | 拉 : 推 ≈ **2 : 1** 或更高 | 恢复内收/外旋拮抗 |
| 一般健身 | 拉 : 推 ≈ **1 : 1 ~ 1.5 : 1** | 维持中立 |
| 力量举偏卧推 | 额外面拉/Band pull-apart | 补后链肩带，不减少卧推 |
