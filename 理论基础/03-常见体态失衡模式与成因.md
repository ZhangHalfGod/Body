# 03 · 常见体态失衡模式与成因

> 综合自原 [02-体态专项-肱骨前移与圆肩](../02-体态专项-肱骨前移与圆肩.md)、[05-肋骨前方肌群](../05-肋骨前方肌群.md)

## 1. 四大失衡模式速览

| 模式 | 视觉特征 | 本质 | 主因（肌肉层面） |
|------|----------|------|------------------|
| **圆肩** | 肩峰前移、胸椎后凸增加 | 整个肩带相对胸廓前移 + 常伴肱骨内旋 | 胸大/胸小短 → 中下斜方/菱形弱 |
| **肱骨前移** | 侧面肱骨头在肩峰前方 | 盂肱关节内肱骨头中心化不足 | 胸大肌内旋 + 肩袖外旋弱 + 后侧链不足 |
| **翼状肩胛** | 推墙时肩胛内侧缘翘起 | 肩胛后侧失去与前侧的平衡 | 前锯肌、下斜方激活不足 |
| **耸肩代偿** | 引体/划船时脖子变短 | 肩胛上提代替了肩胛内收/上旋 | 上斜方过度、下斜方/前锯未参与 |

## 2. 圆肩（Rounded Shoulders）深度剖析

### 2.1 肌肉失衡

| 过紧/过度活跃 | 不足/抑制 |
|----------------|-------------|
| 胸大肌、胸小肌 | 中下斜方肌、菱形肌 |
| 三角肌前束 | 三角肌后束、肩袖外旋肌群 |
| 上斜方肌（代偿性） | 前锯肌、下斜方肌 |
| 背阔肌（内旋肱骨） | 深层颈屈肌（若伴头前伸） |

### 2.2 行为与训练成因

- 长时间伏案、手机、驾驶 → 肩屈 + 内旋位固化
- 训练偏**推 > 拉**（卧推、双杠、飞鸟过多）
- 划船时**耸肩 + 身体后仰**，目标肌群未参与
- 忽视**外旋**与**肩胛后倾**训练

### 2.3 胸廓与呼吸因素

- 胸式呼吸为主、肋廓前侧抬高 → 胸小肌、肋间肌紧张，肩胛基底不稳
- 详见 [04-呼吸力学与体态关联](./04-呼吸力学与体态关联.md)

## 3. 肱骨前移与圆肩的区别

| 维度 | 圆肩 | 肱骨前移 |
|------|------|----------|
| **视觉** | 肩峰向前、手臂内旋 | 肱骨头在肩峰前方（"球"跑出"窝"） |
| **本质** | 肩带整体前移 | 盂肱关节对位问题 |
| **核心弱肌** | 中下斜方、菱形 | 肩袖（尤其冈下肌、小圆肌）、后束 |
| **常共存** | ✅ | ✅ |
| **纠正侧重** | 肩胛内收+后倾 | 外旋+肱头中心化 |

```
  正常侧面观          圆肩                 肱骨前移（夸张示意）

   肩峰                 肩峰→                肩峰
    |                    \                    |
  肱骨头               肱骨头→              肱骨头→（偏前）
```

## 4. 翼状肩胛（Scapular Winging）

| 类型 | 原因 | 特征 |
|------|------|------|
| **前锯肌弱型** | 前锯激活不足 | 肩胛内侧缘+下角翘起（最常见） |
| **下斜方弱型** | 下斜方无法后倾肩胛 | 肩胛下角翘起 |
| **胸小肌过紧型** | 胸小肌短→肩胛前倾 | 喙突区紧张、肩峰前顶 |

## 5. 耸肩代偿链

```
目标：菱形肌 / 中下斜方
         ↓
实际发生：
  上斜方耸肩 ──→ 提肩胛肌
  背阔肌主导 ──→ 肘贴身体猛拉
  腰/伸肌过伸 ──→ 假幅度
  肱二头 ──→ 先酸的是手臂
```

## 6. 失衡的恶性循环

```
浅胸式呼吸（肋前上抬）
        ↓
胸小肌、上斜方、SCM 紧张
        ↓
肩胛前倾 + 内旋 + 难沉肩
        ↓
圆肩 / 肱骨前移 / 翼状肩胛
        ↓
中下斜方、前锯肌更难激活
        ↓
训练中代偿加重 → 回到起点
```
