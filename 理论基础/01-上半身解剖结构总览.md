# 01 · 上半身解剖结构总览

> 来源：原 [01-上半身体系概览.md](../01-上半身体系概览.md)

## 1. 核心结构：三块骨 + 一个关节复合体

上半身（肩带以上）的功能，本质是 **肱骨（上臂）— 肩胛骨 — 锁骨 — 胸廓** 之间的协调。

```
        锁骨 ───────────── 胸骨柄
          \               /
           \   肩胛骨    /
            \  (浮于肋廓) /
             \_________/
                  │
               盂肱关节
                  │
               肱骨头
```

### 1.1 关键结构

| 结构 | 角色 |
|------|------|
| **盂肱关节** | 球窝关节，负责肩屈伸、外展、旋转；活动度大，稳定性依赖周围肌肉 |
| **肩胛胸廓关节** | 非真正关节，肩胛在肋廓上滑动；**约 1/3 抬臂动作来自肩胛上旋** |
| **肩锁关节 / 胸锁关节** | 连接肩带与躯干，传递力量与维持姿态 |

### 1.2 肩胛骨的可动性

肩胛骨有 **6 个自由度** 的运动：

| 运动 | 方向 | 关键肌肉 |
|------|------|----------|
| **上提 / 下压** | 上下垂直 | 上斜方（提）/ 下斜方（压） |
| **内收 / 前伸** | 靠近/远离脊柱 | 菱形肌、中斜方（收）/ 前锯肌（伸） |
| **上旋 / 下旋** | 关节盂朝上/朝下 | 前锯+下斜方（上旋）/ 菱形+提肩胛（下旋） |
| **前倾 / 后倾** | 下角朝前/朝后 | 胸小肌（前倾）/ 下斜方（后倾） |

## 2. 肌肉分层：谁在前、谁在后

### 2.1 浅层（常过度紧张）

| 肌肉 | 主要动作 | 体态风险 |
|------|----------|----------|
| **胸大肌** | 内收、内旋肱骨 | 过紧 → 肱骨内旋前移、圆肩 |
| **三角肌前束** | 肩屈、内旋 | 与胸大肌协同，易「抢戏」 |
| **上斜方肌** | 肩胛上提 | 久坐、stress 常紧张，不等于「中下斜方弱」 |
| **背阔肌** | 肩伸、内收、内旋 | 过紧限制肩胛上旋 |

### 2.2 深层 / 稳定层（常不足）

| 肌肉 | 主要动作 | 重要性 |
|------|----------|--------|
| **中下斜方肌** | 肩胛内收、下压、上旋辅助 | 肩胛稳定的核心 |
| **菱形肌** | 肩胛内收、下旋 | 与上斜方/提肩胛拮抗 |
| **前锯肌** | 肩胛外旋、上旋 | 翼状肩胛的关键拮抗 |
| **肩袖（冈下肌、小圆肌等）** | 肱骨头中心化、外旋 | 盂肱关节稳定 |

### 2.3 肋骨前方肌群

| 肌肉 | 主要动作 | 体态影响 |
|------|----------|----------|
| **胸小肌** | 下拉肩胛喙突 | 短缩 → 肩胛前倾、圆肩、胸廓出口风险 |
| **前锯肌** | 前伸+上旋肩胛 | 弱 → 翼状肩胛 |
| **肋间肌** | 辅助呼吸 | 紧张 → 肋骨上抬、胸式呼吸加重 |

## 3. 关键比例与拮抗对

| 拮抗对 | 平衡状态 | 失衡表现 |
|--------|----------|----------|
| **胸大肌 ↔ 中下斜方+菱形** | 肩胛中性位 | 圆肩、肱骨内旋 |
| **前锯肌 ↔ 菱形肌** | 肩胛贴胸廓 | 翼状 vs 内收过紧 |
| **上斜方 ↔ 下斜方** | 肩胛不上提 | 耸肩代偿 |
| **三角前束 ↔ 后束+肩袖** | 肱头中心 | 肱骨前移 |
| **膈肌 ↔ 胸小肌+肋间** | 肋廓三维扩张 | 胸式呼吸、肩胛基底不稳 |
