# 02 · 体态专项：肱骨前移与圆肩

## 1. 定义与区别

### 圆肩（Rounded Shoulders）

- **视觉**：肩峰向前、胸椎后凸增加、手臂自然下垂时掌心更朝后或朝身体。
- **本质**：整个**肩带**相对胸廓前移 + 常伴**肱骨内旋**。
- **肩胛**：常呈前倾、内旋、轻度下旋或上旋不足。

### 肱骨前移（Anterior Humeral Glide / Forward Head of Humerus）

- **视觉**：侧面观，肱骨头明显在**肩峰前方**；有时肩带位置尚可，但「球」跑出了「窝」的中心。
- **本质**：盂肱关节内**肱骨头中心化不足**，常与肩袖（尤其冈下肌、小圆肌）及后束三角肌无力有关。
- **与圆肩关系**：二者常共存，但可单独存在；纠正动作侧重点略有不同。

```
  正常侧面观          圆肩                 肱骨前移（夸张示意）

   肩峰                 肩峰→                肩峰
    |                    \                    |
  肱骨头               肱骨头→              肱骨头→（偏前）
```

## 2. 成因（多因素叠加）

### 2.1 肌肉失衡

| 过紧 / 过度活跃 | 不足 / 抑制 |
|----------------|-------------|
| 胸大肌、胸小肌 | 中下斜方肌、菱形肌 |
| 三角肌前束 | 三角肌后束、肩袖外旋肌群 |
| 上斜方肌（代偿性） | 前锯肌、下斜方肌 |
| 背阔肌（内旋肱骨） | 深层颈屈肌（若伴头前伸） |

### 2.2 行为与训练

- 长时间伏案、手机、驾驶 → 肩屈 + 内旋位固化
- 训练偏 **推 > 拉**（卧推、双杠、飞鸟过多）
- 划船时**耸肩 + 身体后仰**，目标肌群未参与
- 忽视 **外旋** 与 **肩胛后倾** 训练

### 2.3 胸廓与呼吸

- **胸式呼吸为主、肋廓前侧抬高** → 胸小肌、肋间肌紧张，肩胛基底不稳
- 详见 [05-肋骨前方肌群](./05-肋骨前方肌群.md)

## 3. 自测（非诊断，仅作参考）

### 3.1 静态观察

- 自然站立侧面拍照：耳—肩峰—大转子是否大致一条线？肩峰是否明显在耳垂前方？
- 背面：肩胛内侧缘是否更「贴」向脊柱中间（内旋）？是否有翼状倾向？

### 3.2 简单功能测试

| 测试 | 做法 | 提示异常 |
|------|------|----------|
| **墙滑（Wall Slide）** | 背贴墙，屈肘 90° 沿墙上下滑 | 腰离开墙、耸肩、手无法贴墙 |
| **外旋耐力** | 屈肘 90° 肩外旋位对抗轻阻力 | 很快力竭或肩峰前顶 |
| **肩胛后缩** | 站立，仅收肩胛（不耸肩） | 只能耸肩、无法内收 |
| **前锯肌** | 墙推，观察肩胛是否贴紧 | 内侧缘/下角翘起 |

## 4. 纠正逻辑（四步框架）

```
① 松解限制 → ② 激活弱链 → ③ 整合模式 → ④ 负荷渐进
```

### ① 松解限制（非替代训练）

- 胸小肌、胸大肌：门框拉伸、网球/球松解（肋骨 2–4 区域）
- 背阔肌：侧位拉伸（内旋位常短）
- 上斜方：若明显紧张，可松解，但**不要只做这个**

### ② 激活弱链

- 中下斜方 + 菱形肌：Y-T-W、面拉、弹力带肩胛后缩
- 前锯肌：墙推 plus、弹力带 protraction
- 肩袖外旋：侧 lying 外旋、古巴旋转（轻重量）

### ③ 整合模式

- 肩胛后倾 + 外旋 + 推/拉 结合：例如 bear crawl、土耳其起立轻量、俯卧撑时「拧地」外旋
- 强调 **「坑」里推/拉**：引体/划船时先收肩胛再动肘

### ④ 负荷渐进

- 拉类动作量 ≥ 推类（见 [06-肌肉动作规划](./06-肌肉动作规划.md)）
- 卧推/推举前确保可完成无耸肩的肩胛设置

## 5. 常见误区

| 误区 | 说明 |
|------|------|
| 「多练背就好了」 | 划船姿势错误仍练背阔/二头，中下斜方依旧没刺激 |
| 「开肩、开胸猛拉」 | 暴力拉伸可能刺激前方结构；应配合激活与呼吸 |
| 「只练后束三角肌」 | 后束重要，但无法替代肩胛内收肌群（菱形肌、中下斜方） |
| 「夹背一直夹着」 | 肩胛内收是**动态能力**，不是静态死夹；需在全幅度动作中控制 |
