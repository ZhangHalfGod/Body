# 04 · 中下斜方肌与菱形肌

## 1. 为什么这两块肌肉「难练到」

### 1.1 解剖与力学劣势

| 因素 | 说明 |
|------|------|
| **纤维方向** | 中下斜方偏水平/斜下，菱形肌斜向脊柱；**大重量垂直拉** 往往先练到背阔肌 |
| **位置深** | 菱形肌位于中下斜方深面，**易被背阔肌、大圆肌、肱二头覆盖** |
| **负荷敏感** | 这两块更吃 **控制与角度**，不是单纯堆重量 |

### 1.2 常见代偿链（为什么你「练背」却练不到它们）

```
目标：菱形肌 / 中下斜方
         ↓
实际发生：
  上斜方耸肩 ──→ 提肩胛肌
  背阔肌主导 ──→ 肘贴身体猛拉
  腰/ext 过伸 ──→ 假幅度
  肱二头 ──→ 先酸的是手臂
```

### 1.3 神经肌肉因素

- 久坐、圆肩 → **抑制** 中下斜方/菱形肌的主动激活（Janda 交叉综合征中「弱链」之一）
- 大脑习惯 **「拉」= 把手拉向身体**，而非 **「先收肩胛再拉」**

## 2. 功能再定义：它们到底干什么

### 中下斜方肌（Middle / Lower Trapezius）

- **中斜方**：肩胛**内收**（向脊柱）
- **下斜方**：肩胛**内收 + 下压 + 上旋**（与上斜方「上提」拮抗）
- **关键场景**：过头举、引体向上半段、游泳、投掷的「收肩」

### 菱形肌（Rhomboid Major / Minor）

- **主要**：肩胛**内收**、**下旋**（与前锯肌上旋形成动态平衡）
- **稳定**：维持肩胛贴胸廓，防止过度外旋/翼状

## 3. 有效激活原则

### 原则 A：低负荷、高意识

- 先用 **无负重或弹力带** 找到「肩胛向脊柱滑动」的感觉
- 指标：**不耸肩、颈不长**

### 原则 B：肩胛先动，肘后动

- 划船类：**先收肩胛 2–3 cm，再屈肘**；离心时**先伸肘，再放肩胛**

### 原则 C：角度匹配纤维

| 目标 | 推荐臂角 / 体姿 |
|------|-----------------|
| **中斜方 + 菱形** | 水平外展约 90°、肘略外展；**Face Pull**、**Band pull-apart** |
| **下斜方** | 臂向前上约 **Y 字**（130° 外展 + 前屈）；**Wall slide**、**Prone Y** |
| **菱形肌（垂直拉）** | 窄握、**胸贴垫** 划船；引体 **胸触杠** 意识在收肩胛 |

### 原则 D：离心与停顿

- 顶峰收缩 **1–2 秒**；离心 **2–3 秒**
- 中下斜方对 **时间 under tension** 反应好于极限重量

## 4. 推荐动作分级

### Tier 1 — 激活（每次练背/拉日前）

| 动作 | 要点 | 量 |
|------|------|-----|
| **Scapular Push-up / Pull-up** | 仅动肩胛，几乎不弯肘 | 2×10–15 |
| **Band Pull-apart** | 水平外展，拇指朝后 | 2×15–20 |
| **Wall Slide** | 沉肩、贴墙、不耸肩 | 2×8–10 |
| **Prone Y-T-W** | 轻重量或自重，**别用上斜方** | 各 2×8–12 |

### Tier 2 — 主训练（中等负荷）

| 动作 | 针对 | 要点 |
|------|------|------|
| **Face Pull**（绳） | 中斜方、菱形、后束 | 外旋位结束、拉向额/耳 |
| **Chest-supported Row** | 菱形、中斜方 | 胸贴垫防借力；窄握 |
| **One-arm DB Row**（肘外展 45°） | 菱形 | 先收肩胛，躯干少转 |
| **Straight-arm Lat Pushdown** 变体 | 下斜方辅助 | 微屈肘，向下滑「压肩」 |

### Tier 3 — 进阶整合

| 动作 | 说明 |
|------|------|
| **引体向上**（全幅度） | 离心先放肩胛；向心先收肩胛 |
| **Inverted Row** | 身体越水平，越偏水平拉 + 肩胛内收 |
| **Farmer Carry**（沉肩走） | 下斜方耐力、抗耸肩 |

## 5. 排雷：这些做法练不到目标

- ❌ 大重量 **俯身杠铃划船 + 身体猛晃**
- ❌ **高位下拉** 拉到颈后、含胸
- ❌ **耸肩** 当「练斜方」（那是上斜方）
- ❌ 没有激活直接上重量 **Seal Row** 仍用背阔拉
- ❌ 夹背「死劲」不呼吸，肋廓僵死（见 05）

## 6. 进度指标（怎么知道练对了）

- 激活后 **Wall slide** 手能更高、腰仍贴墙
- 划船时 **上斜方不先酸**；酸感在 **肩胛内侧 / 腋下后方**
- 引体 **耸肩减少**，肩峰不适感下降
- 静态 **肩胛内收** 可维持 10–15 秒且不耸肩
