# 06 · 肌肉动作规划（三分化 · 腿每周一次）

## 1. 规划原则总结

| 原则 | 具体做法 |
|------|----------|
| **三分化** | 推 / 拉 / 腿，**每周各 1 次**（共 3 练） |
| **拉 > 推** | 圆肩/办公族：拉类组数约为推的 **1.2–1.5 倍**（腿日不计入） |
| **先激活后主训** | 推日、拉日前 5–8 分钟肩胛激活；腿日做髋踝热身 |
| **先松后练**（可选） | 推/拉日前胸小肌球松 1–2 分钟 |
| **动作顺序** | 激活 → 主复合 → 辅孤立 → 收尾（面拉/外旋 或 核心） |
| **质量优先** | 中下斜方/菱形肌：**轻重量 + 停顿 + 肩胛领先** |
| **腿日一次够** | 一次覆盖蹲、铰链、单腿；不拆成两次腿日 |

## 2. 推荐周结构：三分化（主推）

适用于：**有圆肩/肱骨前移倾向**、希望 **上身纠正 + 下肢维持** 的训练者。

| 练次 | 主题 | 部位 | 时长参考 |
|------|------|------|----------|
| **Day 1 · 推** | 胸肩三头 + 肩带纠正 | 胸、肩、三头 | 55–70 min |
| **Day 2 · 拉** | 背二头 + 肩胛稳定 | 背、二头、中下斜方/菱形 | 60–75 min |
| **Day 3 · 腿** | 下肢 + 核心 | 股四、臀腿后、小腿、核心 | 55–70 min |

### 建议排布（示例）

| 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 |
|------|------|------|------|------|------|------|
| **推** | 休 | **拉** | 休 | **腿** | 休 | 休 |

也可 **推 → 休 → 拉 → 休 → 腿**，中间至少隔 **1 天** 再练腿（尤其拉日后背疲劳时）。

### 每周组数参考

| 类型 | 组数/周 | 说明 |
|------|---------|------|
| **拉**（背+二头） | 16–22 | 含划船、引体/下拉 |
| **推**（胸+肩+三头） | 12–18 | 卧推、推举、辅项 |
| **肩带纠正** | 12–16 | 面拉、外旋、Pull-apart、Y-T-W |
| **腿** | 14–20 | 蹲 4–6 组 + 铰链 4–6 组 + 单腿 3–4 组 |
| **核心** | 6–10 | 可放在腿日或推/拉日收尾 |

> 肩带纠正量 **不减少**：腿只练一次，省出的时间留给面拉/激活，而不是多加胸肌组数。

## 3. 单次训练模板

### 3.1 Day 1 · 推日（胸肩三头 + 少恶化圆肩）

```
① 热身 5 min（摆臂、弹力带绕肩）
② 激活（必做）
   - 弹力带外旋                         2×15
   - 肩胛后缩（沉肩，不耸肩）           2×10
③ 主项
   - 卧推 / 哑铃卧推                    3–4×6–10
   - 过头推举 / 哑铃肩推（适中重量）    3×6–10
④ 辅项
   - 上斜哑铃推 / 飞鸟（幅度可控）      2–3×10–12
   - 侧平举（轻）                       2×12–15
   - 三头下压 / 窄握俯卧撑              2–3×10–12
⑤ 收尾（必做）
   - Face Pull                          3×12–15
   - 肩外旋（侧躺或绳）                 2×15
⑥ 可选
   - 胸大肌/胸小肌拉伸                  30s/侧
```

### 3.2 Day 2 · 拉日（背二头 + 中下斜方/菱形）

```
① 热身 5 min（摆臂、弹力带外旋）
② 激活（必做）
   - Scapular Pull-up 或 Band 肩胛后缩  2×12
   - Band Pull-apart                    2×15
   - Prone Y 或 Wall Slide              2×10
③ 主项
   - 引体向上 / 高位下拉（先收肩胛）    3–4×6–10
   - 胸贴垫划船 / 杠铃划船（先收肩胛）  3–4×6–10
④ 辅项
   - 单臂哑铃划船（肘外展约 45°）       3×10/侧
   - 直臂下压（下斜方沉肩意识）         2×12
   - 二头弯举                           2–3×10–12
⑤ 收尾
   - Face Pull                          2–3×12–15
⑥ 可选
   - 胸小肌球松 + 拉伸                  1 min
```

### 3.3 Day 3 · 腿日（每周一次 · 蹲 + 铰链 + 单腿）

```
① 热身 8–10 min
   - 泡沫轴或动态拉伸：髋屈肌、臀、腘绳肌
   - 空杆或自重深蹲 2×10
   - 臀桥 2×12（激活臀大肌，减轻腰代偿）
② 主项 · 蹲类（股四 + 臀）
   - 后蹲 / 前蹲 / 高脚杯深蹲           3–4×6–10
   - 或 腿举 / 史密斯深蹲（按器械条件）  3×8–12
③ 主项 · 铰链类（臀 + 腘绳肌）
   - 罗马尼亚硬拉 / 直腿硬拉            3–4×8–10
   - 或 臀推 / 早安式（轻–中重量）      3×8–12
④ 辅项 · 单腿（平衡 + 补弱侧）
   - 保加利亚分腿蹲 / 弓步蹲            3×8–10/腿
   - 或 腿弯举（器械）                  3×10–12
⑤ 小腿（可选）
   - 提踵（站姿或坐姿）                 3×12–15
⑥ 核心收尾
   - 死虫 / 鸟狗 / 平板支撑             2–3 组
   - 或 农夫行走（沉肩）2×30–40 m      ← 顺带练下斜方耐力
```

**腿日注意（与上身体态相关）**

- 深蹲时避免 **圆背 + 胸廓塌陷**；挺胸肋、收核心，与 [05-肋骨前方肌群](./05-肋骨前方肌群.md) 的呼吸一致。
- 大重量深蹲日 **不必** 做肩带激活，但 **不要** 在肩峰不适时强行颈后深蹲。
- 腿日 **不替代** 拉日的肩胛训练；圆肩纠正仍靠 Day 1/2 的面拉与激活。

### 3.4 纠正微训练（10 分钟，非腿日也可做）

适合办公间隙，**不追求力竭**：

| 顺序 | 动作 | 量 |
|------|------|-----|
| 1 | 胸小肌球松 | 30s/侧 |
| 2 | 90-90 呼吸或侧向肋呼吸 | 5 次 |
| 3 | Wall Slide | 2×8 |
| 4 | Band Pull-apart | 2×15 |
| 5 | Push-up Plus | 2×12 |

建议安排在 **推日/拉日前**，或休息日；腿日当天可只做呼吸 + 髋部拉伸。

## 4. 动作-肌肉对照表

### 4.1 上身 · 肩带与体态

| 目标肌肉 | 首选动作 | 避免 |
|----------|----------|------|
| **中下斜方** | Prone Y、Wall slide、Face pull 沉肩段 | 耸肩划船 |
| **菱形肌** | Chest-supported 窄握划船、Inverted row | 借力俯身杠铃拉 |
| **前锯肌** | Push-up plus、Scapular push-up | 只夹背不练 protraction |
| **肩袖外旋** | 侧躺外旋、古巴旋转 | 颈后高拉 |
| **胸小肌松解** | 球压 + 拉伸 | 只拉不松 |

### 4.2 下肢 · 腿日一次练全

| 目标 | 首选动作 | 说明 |
|------|----------|------|
| **股四** | 后蹲、前蹲、腿举、分腿蹲 | 蹲类主项选 1–2 个即可 |
| **臀大肌** | 臀推、RDL、弓步 | 铰链与单腿都会练到 |
| **腘绳肌** | RDL、腿弯举、北欧腿弯举（进阶） | 与蹲类互补 |
| **小腿** | 站姿/坐姿提踵 | 时间紧可隔周做 |
| **核心** | 死虫、鸟狗、农夫走 | 抗伸展 + 肩沉（农夫走） |

## 5. 进阶与周期（4 周示例）

| 周 | 推/拉 | 腿 | 调整 |
|----|-------|-----|------|
| **1** | 找肩胛感觉；重量 **~70%** | 动作模式为主，组数偏少 | 激活量可加倍 |
| **2** | 拉类 +1 组；Face pull 略加重 | 蹲/拉各 +1 组 | 仍控动作质量 |
| **3** | 引体/划船 **先收肩胛** 全幅度 | 单腿项加 1 组或加重 | — |
| **4** | 减载：组数 **−25~30%** | 腿日主项组数 −1~2 | 保留激活与面拉 |

每 4 周复测 [02](./02-体态专项-肱骨前移与圆肩.md) 中的 Wall slide、静态肩胛内收。

## 6. 是否合理？——自检清单

**上身（推/拉日）**

- [ ] 划船/引体时 **可以先动肩胛再动肘**
- [ ] Face pull **不耸肩**
- [ ] 每周 **拉 ≥ 推**（组数）
- [ ] 中下斜方训练日 **上斜方不先力竭**

**下肢（腿日）**

- [ ] 深蹲/硬拉 **腰不主导**（可先降重量练臀桥激活）
- [ ] 一次腿日包含 **蹲 + 铰链 + 单腿** 三类中的至少各 1 个动作
- [ ] 腿日后 **至少休 1–2 天** 再安排大强度拉或蹲

若上身项多数未达标，**先维持 2–3 周 Tier 1 激活**，再加重推/拉主项。

## 7. 与体态专项的衔接

```
日常微纠正（10 min，推/拉休息日）
        +
每周 3 练：推 → 拉 → 腿（腿仅 1 次）
        +
推/拉技术：肩胛领先、面拉/外旋收尾
        ↓
4–8 周评估：圆肩/肱骨前移 + 下肢力量/酸痛
```

若 8–12 周肩带无改善且伴疼痛 → 建议物理治疗评估。

## 8. 备选：想多加一次上身时

仍保持 **腿每周 1 次**，可把第 4 次定为 **弱项上身**（20–30 min），不增加腿：

| 加练日 | 内容 |
|--------|------|
| **弱项拉** | 仅 Y-T-W + Face Pull + 轻划船 2 组 |
| **弱项推** | 仅外旋 + Band pull-apart + 轻后束 |

**不要** 变成一周两练腿，除非恢复很好且目标转向下肢专项。
