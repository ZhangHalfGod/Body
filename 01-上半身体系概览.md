# 01 · 上半身体系概览

## 1. 核心结构：三块骨 + 一个关节复合体

上半身（肩带以上）的功能，本质是 **肱骨（上臂）— 肩胛骨 — 锁骨 — 胸廓** 之间的协调。

```
        锁骨 ───────────── 胸骨柄
          \               /
           \   肩胛骨    /
            \  (浮于肋廓) /
             \_________/
                  │
               盂肱关节
                  │
               肱骨头
```

### 关键概念

| 结构 | 角色 |
|------|------|
| **盂肱关节** | 球窝关节，负责肩屈伸、外展、旋转；活动度大，稳定性依赖周围肌肉 |
| **肩胛胸廓关节** | 非真正关节，肩胛在肋廓上滑动；**约 1/3 抬臂动作来自肩胛上旋** |
| **肩锁关节 / 胸锁关节** | 连接肩带与躯干，传递力量与维持姿态 |

## 2. 肌肉分层：谁在前、谁在后

### 浅层（常过度紧张）

- **胸大肌**：内收、内旋肱骨；过紧 → 肱骨内旋前移、圆肩
- **三角肌前束**：肩屈、内旋；与胸大肌协同，易「抢戏」
- **上斜方肌**：肩胛上提；久坐、Stress 常紧张，不等于「中下斜方弱」

### 深层 / 稳定层（常不足）

- **中下斜方肌**：肩胛内收、下压、上旋辅助
- **菱形肌**：肩胛内收、下旋（与上斜方/提肩胛肌拮抗）
- **前锯肌**：肩胛外旋、上旋；「翼状肩胛」常与此相关
- **肩袖（冈下肌、小圆肌等）**：肱骨头中心化，外旋

### 肋骨前方

- **胸小肌**：连接肋骨 3–5 与肩胛喙突；短缩 → 肩胛前倾、圆肩、压迫神经血管（胸廓出口）
- **肋间肌 / 膈肌**：呼吸时改变胸廓容积，影响肋骨位置与肩胛基底

## 3. 功能链：抬手时发生了什么

健康抬臂（约 0°→180°）的典型分工：

1. **0°–60°**：主要盂肱关节运动
2. **60°–120°**：肩胛开始明显上旋（前锯肌 + 下斜方）
3. **120°–180°**：盂肱 + 肩胛协同；若中下斜方/前锯不足 → 上斜方代偿、耸肩

**肩胛骨应有的运动**：上旋、后倾（轻微）、适度外旋（远离脊柱）。

## 4. 常见失衡模式（上半部分）

| 模式 | 表现 | 常见原因 |
|------|------|----------|
| **圆肩** | 肩峰前移、胸廓后缩感 | 胸大肌/胸小肌短、中下斜方/菱形肌弱 |
| **肱骨前移** | 侧面看肱骨头在肩峰前方 | 胸大肌内旋 + 肩袖外旋弱 + 后侧链不足 |
| **翼状肩胛** | 推墙时肩胛内侧缘翘起 | 前锯肌、下斜方激活不足 |
| **耸肩代偿** | 引体/划船时脖子短、肩一直提着 | 上斜方过度、下斜方/前锯未参与 |

## 5. 训练原则（上半部分）

1. **先恢复活动度，再谈力量**：胸小肌、胸大肌、背阔肌过紧会限制肩胛后倾与肱骨外旋。
2. **先激活后负荷**：中下斜方、菱形肌、前锯肌需要在主训练前「被找到」。
3. **推拉比例**：长期只练推（卧推、飞鸟）而不练拉（划船、面拉），圆肩风险上升。
4. **动作质量 > 重量**：划船用背阔肌和肱二头代偿，练不到菱形肌/中下斜方。
