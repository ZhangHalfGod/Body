# 身体结构文档 · 上半部分

本目录聚焦**上半身**的解剖关系、常见体态问题与训练规划，当前重点覆盖：

- 胸大肌、三角肌对肩带与肩胛骨的影响
- 肱骨前移、圆肩等专项体态问题
- 中下斜方肌、菱形肌（较难激活与训练）
- 肋骨前方肌群（胸小肌、前锯肌、肋间肌等）与呼吸/肩带的关系

## 使用说明

1. 先读 **01 概览**，建立肩带整体模型。
2. 若有圆肩/肱骨前移，重点读 **02 专项** 与 **05 肋骨前方**。
3. 训练设计以 **06 肌肉动作规划** 为主，配合 **03、04** 理解「为什么要这样练」。

> 本文档为结构与训练参考，不替代医疗评估。若存在明显疼痛、麻木或活动受限，请先咨询医生或物理治疗师。
