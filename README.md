- 肋骨前方肌群（胸小肌、前锯肌、肋间肌等）与呼吸/肩带的关系

## 📖 使用说明

### 推荐阅读路径

| 目的 | 路径 |
|------|------|
| **系统学习** | 理论基础 01→02→03→04 → 实践基础 01→02→03→04 → 训练状态 01→02→03→04 |
| **快速纠正体态** | 实践基础 01（自测）→ 实践基础 02（松解）→ 实践基础 03（激活）→ 训练状态 02（计划） |
| **优化训练计划** | 训练状态 01（器械）→ 训练状态 02（计划）→ 训练状态 03（覆盖度分析）→ 训练状态 04（周期化） |
| **查阅特定肌肉** | 理论基础 01（解剖）→ 实践基础 03（激活）→ 训练状态 02（计划中的对应动作） |

### 每条路径说明

1. **系统学习**：先建立整体知识框架，再进入实操，最后优化训练。
2. **快速纠正体态**：直接自测→松解→激活→执行计划，适合已有基础概念的读者。
3. **优化训练计划**：从现有器械出发，调整计划的覆盖度与进阶节奏。
4. **查阅特定肌肉**：想了解某块肌肉时，解剖→激活→训练计划三站查完。
